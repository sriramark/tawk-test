To test the chat feature, please click on the icon located at the bottom corner.

Once the Sunonaa website is set up, the chat feature will be customized specifically for your needs. A new and fresh admin panel will be created, using Sunonaa's workspace account or the email address of your choice.

For testing purposes, I have integrated the chat feature into this blank page.
To access the live chat functionality, navigate to the admin dashboard and sign in using credentials sent to you. From there, you can view and respond to chat messages, create agents to handle the chats, create chat groups, and much more.

Please note that the chat history will be stored in the user's browser, allowing their previous chats to be loaded when they visit the website again.
